# Politica de Facturacion Electronica

Northwind ofrece facturacion electronica para todos sus clientes.

## Requisitos

- Clave de acceso y Certificado Digital
- Registro en el sistema de facturacion

## Proceso

1. Ingresa al modulo de facturacion
2. Selecciona "Factura Electronica"
3. Completa los datos del cliente
4. Genera y envía la factura
